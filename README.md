# The-research-project
Project Description This research project investigates SpaceX launch records to understand patterns in mission success. The dataset contains information about launch sites, payload masses, booster versions, and whether each mission succeeded or failed. By analyzing these variables, the project aims to uncover how different factors contribute .
The work involves:

Comparing launch sites to see which locations have higher success rates.

Studying payload ranges to determine whether heavier or lighter payloads influence outcomes.

Examining booster categories to identify technological factors linked to success.

Purpose
The goal is to provide insights into operational efficiency and reliability in modern spaceflight. This type of analysis contributes to broader astronomy and aerospace research by highlighting how engineering choices and logistical factors impact the success of missions that expand human presence in space.
